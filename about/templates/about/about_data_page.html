Stock Data

Real Estate Data

This data comes from Quandl, specifically the Zillow data set. It is used for gathering all data on any real estate areas that you would like. To use it, you have to reference the data set Here.

Once you are at the dataset please type into the search bar the area you would like to reference. Afterwards, find the code that relates to that area, which is shown in a red rectangle to the right on this page.

Once you have found the right key, copy and paste this key back into the correlation form.



Macroeconomic Data

This data comes from Quandl, specifically the Yale research data set. It is used in gathering myriad economic and macro economic indicators including CPI and index prices. To use it, you have to reference the data set Here.

Once you are at the dataset please type into the search bar the indicator you would like to reference. Afterwards, find the code that relates to that area, which is shown in a red rectangle to the right on this page.

Once you have found the right key, copy and paste this key back into the correct form.
