User Guide

Get Data

Get Stock Data

This is the very first option you have under home in the navigation bar. What this function will do for you is grab some sort of data, , be it economic/real esate/stock data. It will also plot a moving average on top of the indicator. Once you click on the get data link, you will be redirected to the first page on the right.

What you will then do is input a ticker/real estate/macroeconomic code. You can find more information on this Here

Once you have found your code, input the code into the form. Make sure you have the correct data type selected. Reference the right image for more help on this.







Regression Analysis

Simple Regression Analysis

This is a simple regression analysis which draws a line made from OLS between the data. What this function will do for you is grab some sort of data, , be it economic/real esate/stock data. If you are not certain how to use a regression, please reference the strategies page Here

Once you click on the get data link, you will be redirected to the form page, which you can see above.

Make sure you fill out the fields. Reference the Get Data paragraph from finding how to fill out this form.

Correlation

Heat Map/Corellation analysis

This is going to create a correlation matrix between the selected indicators/stocks/data. You can grap macroeconomic/real estate data and also ticker data. If you are unfamiliar with how a correlation matrix works, please reference the strategies page Here

When you have arrived at the form page, make sure you enter the data into the date and first two inputs(2), referenced to the right. Make sure that you have also clicked the right data type.

If you would like to correlate up to 4 inputs, please make sure to enter the data, aswell as clicking on the correct data type.

The other two inputs are set to default none, so please change them if you would like to use those inputs.
